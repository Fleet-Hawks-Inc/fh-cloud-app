Roles and Permissions Descriptions
Organization Admin: Organization admin have the ability to make any and all configuration changes throughout the entire organization. And only organization admin can view driver pays, SIN numbers and profite & loss of company.
Super Admin: Super admin have the ability to edit the fleet hawks dashboard and make non-financial configuration changes for the entire organization.
View Only Admin: View only admin have the ability to view everything across all tags in the organization but are unable to make any configuration changes.
Dispatch Manager: Dispatch admin have ability to access full dispatch module of the organization.
Safety Manager: Safety admin have ability to access full safety module of the organization.
Accounting Manager: Accounting admin have ability to access full accounting module of the organization.
Technicians: Technicians admin have ability to access full maintenance and inventory module of the organization.
